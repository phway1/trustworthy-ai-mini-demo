# Trustworthy AI Mini Demo

This repository presents a demonstration of model evaluation and bias-aware analysis from a Trustworthy AI perspective.

## Purpose

The purpose of this project is to show how AI systems should be evaluated not only by overall performance, but also by subgroup-level performance, fairness concerns, documentation, and lifecycle monitoring.

## Contents

- `model_evaluation_demo.ipynb`: Basic machine learning model evaluation using accuracy, precision, recall, F1-score, and confusion matrix.
- `bias_analysis_demo.ipynb`: Simple subgroup analysis to show how aggregate performance can hide group-level harms.
- `trustworthy_ai_checklist.md`: A short lifecycle checklist for Trustworthy AI assessment.

## Key Concepts

- Trust vs. trustworthiness
- Model performance evaluation
- Bias and subgroup analysis
- Data quality and governance
- Human oversight
- Monitoring and re-validation

## Reflection

My background is in Educational Big Data Analytics and machine learning education. Through this mini-project, I aim to connect empirical data analysis with Trustworthy AI principles such as fairness, transparency, accountability, and responsible data use.
