# Trustworthy AI Lifecycle Checklist

## 1. Planning Stage

Key activities:

- Define the purpose of the AI system.
- Identify stakeholders and affected groups.
- Assess possible risks to individuals, groups, and society.
- Decide whether AI is appropriate for the problem.
- Define success criteria beyond accuracy.

Trustworthiness focus:

- Accountability
- Human oversight
- Risk management
- Social impact

## 2. Data Stage

Key activities:

- Check data source and collection method.
- Evaluate data quality.
- Check missing values, duplicates, and inconsistent formats.
- Assess representativeness of the dataset.
- Identify possible technical and contextual bias.
- Document data limitations.

Trustworthiness focus:

- Data quality
- Fairness
- Privacy
- Traceability
- Transparency

## 3. Model Development Stage

Key activities:

- Select appropriate model.
- Train and validate the model.
- Evaluate performance using suitable metrics.
- Compare subgroup-level performance.
- Check possible bias and robustness issues.
- Document model assumptions and limitations.

Trustworthiness focus:

- Reliability
- Bias diagnosis
- Robustness
- Explainability
- Reproducibility

## 4. Deployment Stage

Key activities:

- Inform users when AI is being used.
- Provide appropriate explanation of AI outputs.
- Define human oversight process.
- Prepare technical documentation.
- Establish incident response process.

Trustworthiness focus:

- Transparency
- Human oversight
- Accountability
- Safety
- User awareness

## 5. Operation and Monitoring Stage

Key activities:

- Monitor model performance after deployment.
- Check data drift and model drift.
- Monitor subgroup-level fairness.
- Collect user feedback and incident reports.
- Re-validate the model periodically.
- Re-train the model when necessary.

Trustworthiness focus:

- Continuous monitoring
- Fairness over time
- Safety
- Reliability
- Lifecycle governance

## 6. Retirement or Update Stage

Key activities:

- Decide whether the model should be updated, replaced, or retired.
- Archive documentation and decision records.
- Review lessons learned.
- Communicate changes to relevant stakeholders.

Trustworthiness focus:

- Accountability
- Documentation
- Risk reduction
- Continuous improvement

## Summary

Trustworthy AI is not achieved only by building an accurate model. It requires lifecycle-wide management from planning to monitoring and retirement.

A trustworthy AI system should be reliable, fair, transparent, explainable, safe, privacy-aware, accountable, and properly governed.
